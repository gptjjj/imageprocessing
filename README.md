**프로젝트 설명**

JavaScript를 기반으로 raw파일과 컬러이미지파일을 구분하여 디지털 영상처리 구현

**주요 기능** 

1) User : 회원가입 , 로그인, MyPage , 쪽지

2) Board :  게시물 CRUD 및 댓글 , 게시물 스크랩(찜기능)

3) Admin :  회원관리, 게시물관리, 일별 데이터(구글 차트)
